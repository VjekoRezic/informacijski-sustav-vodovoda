# Informacijski sustav Vodovoda X - web application
Made with Vue.js

## Project setup
1. Clone repository.
```
git clone https://github.com/antekvesic/pis.git
```
2. Change directory.
```
cd pis/web-app/
```
3. Install dependencies.
```
npm install
```
4. Run development server.
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```
